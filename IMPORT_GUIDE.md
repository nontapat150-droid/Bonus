# 📊 คู่มือการนำเข้าสินค้า (Import Guide)

## ✅ รูปแบบไฟล์ที่ถูกต้อง

### คอลัมน์ที่จำเป็น (บังคับ)
| คำสั่ง | ตำแหน่ง | ตัวอย่าง | หมายเหตุ |
|--------|--------|---------|---------|
| ชื่อสินค้า | คอลัมน์ที่ 1 (A) | iPhone 15 Pro | ต้องมีค่า |
| รุ่น / Model | คอลัมน์ที่ 2 (B) | A3001 | ต้องมีค่า |
| ซีเรียล / SN | คอลัมน์ที่ 3 (C) | SN12345678 | ตัวเลือก - ถ้าว่างจะสร้างอัตโนมัติ |
| รหัสสินค้า (ตัวเลือก) | คอลัมน์ที่ 4 (D) | PROD-001 | ตัวเลือก |

---

## 📝 ตัวอย่างไฟล์ที่ถูกต้อง

### ตัวอย่าง 1: ไฟล์ Excel (ที่นิยม)
```
ชื่อสินค้า              รุ่น (Model)    ซีเรียล (SN)    หมายเหตุ
iPhone 15 Pro           A3001           SN12345678      จากซัพพลายเออร์ A
Samsung Galaxy S24      SM-S921B        SN87654321      จากซัพพลายเออร์ B
MacBook Pro             MBP16-2024      SN11111111      ครื่องใหม่
iPad Air                iPad-Air-6      SN22222222      (ถ้าว่าง = สร้างอัตโนมัติ)
Dell Monitor            U2724DE                         SN สร้างอัตโนมัติ
```

### ตัวอย่าง 2: ไฟล์ CSV (ถ้า Excel ไม่ได้)
```csv
ชื่อสินค้า,รุ่น (Model),ซีเรียล (SN),หมายเหตุ
iPhone 15 Pro,A3001,SN12345678,จากซัพพลายเออร์ A
Samsung Galaxy S24,SM-S921B,SN87654321,จากซัพพลายเออร์ B
```

## ❌ ข้อผิดพลาดที่พบบ่อย

### 1. ข้อมูลไม่สมบูรณ์
**❌ ผิด:**
```
ชื่อสินค้า              รุ่น (Model)    ซีเรียล (SN)
iPhone 15 Pro                           SN12345678
                        A3001           SN87654321
```
**✅ ถูก:**
```
ชื่อสินค้า              รุ่น (Model)    ซีเรียล (SN)
iPhone 15 Pro           A3001           SN12345678
Samsung Galaxy          SM-S921B        SN87654321
```

### 2. ลำดับคอลัมน์ผิด
**❌ ผิด:**
```
รุ่น (Model)    ชื่อสินค้า              ซีเรียล (SN)
A3001           iPhone 15 Pro           SN12345678
```
**✅ ถูก:**
```
ชื่อสินค้า              รุ่ง (Model)    ซีเรียล (SN)
iPhone 15 Pro           A3001           SN12345678
```

### 3. เยื้องหรือมีช่องว่างด้านหน้า/หลัง
**❌ ผิด:**
```
  iPhone 15 Pro   |   A3001   |  SN12345678  
```
**✅ ถูก:**
```
iPhone 15 Pro|A3001|SN12345678
```

### 4. ใช้รูปแบบไฟล์ไม่ถูกต้อง
**❌ ผิด:**
- PDF
- Word (.docx)
- Google Sheets (ต้องดาวน์โหลดเป็น Excel/CSV)

**✅ ถูก:**
- Excel (.xlsx, .xls)
- CSV (.csv) - ถ้า Excel ไม่ได้

## 🔧 การแก้ไขปัญหา

### ถ้าเห็น: "ไม่พบข้อมูลที่ถูกต้องในไฟล์"

1. **ตรวจสอบคอลัมน์แรก (ชื่อสินค้า)**
   - ต้องมีข้อมูลในทุกแถว
   - ไม่ควรมีแถวว่างระหว่างข้อมูล

2. **ตรวจสอบคอลัมน์ที่สอง (รุ่น)**
   - ต้องมีข้อมูลในทุกแถว
   - ต้องมีค่าเลยไม่อย่างนั้นจะข้าม

3. **ตรวจสอบรูปแบบไฟล์**
   - ปิดไฟล์ Excel ทั้งหมด
   - บันทึกไฟล์ใหม่เป็น .xlsx

4. **ลองดาวน์โหลด Template**
   - คลิกปุ่ม "ดาวน์โหลด Template"
   - ใช้เป็นแม่แบบ
   - กรอกข้อมูลของคุณ
   - บันทึกและนำเข้า

## 💡 เคล็ดลับ

### ✓ วิธีสร้างไฟล์ Excel ที่ถูกต้อง

**ขั้นตอน 1:** เปิด Excel
```
New Workbook
```

**ขั้นตอน 2:** สร้างแถว Header (ไม่บังคับ แต่แนะนำ)
```
A1: ชื่อสินค้า
B1: รุ่น (Model)
C1: ซีเรียล (SN)
D1: หมายเหตุ
```

**ขั้นตอน 3:** ใส่ข้อมูล
```
A2: iPhone 15 Pro
B2: A3001
C2: SN12345678
D2: ข้อความอื่น ๆ

A3: Samsung Galaxy S24
B3: SM-S921B
C3: SN87654321
```

**ขั้นตอน 4:** บันทึก
```
File → Save As
ประเภท: Excel Workbook (.xlsx)
ชื่อ: inventory_import.xlsx
```

**ขั้นตอน 5:** นำเข้า
```
1. เปิดระบบคลังสินค้า
2. ไป Tab "นำเข้า"
3. คลิก "เลือกไฟล์ Excel"
4. เลือก inventory_import.xlsx
5. ยืนยันการนำเข้า
```

## 📞 ติดต่อสนับสนุน

ถ้าปัญหายังคงมีอยู่:
1. ตรวจสอบ Browser Console (F12) เพื่อดูข้อมูลข้อผิดพลาด
2. เปิด Developer Tools (F12) → Console
3. ลองใหม่และบันทึกข้อความ Error
4. ติดต่อผู้ดูแลระบบ
